# Week 5 – CRUD App with MongoDB

## 📌 Features:
- Create a new user
- View all users
- Update user details
- Delete a user

## ⚙️ Tech Stack:
- Node.js
- Express.js
- MongoDB
- Mongoose
- dotenv

## 📦 How to Run:
1. Run `npm install`
2. Make sure MongoDB is running locally
3. Set up `.env` file
4. Run: `node index.js`
5. Test routes via Postman or Thunder Client (VS Code)
